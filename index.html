Windows Phone Helpers

Some C# helpers for wp7 applications

Download this project as a .zip file Download this project as a tar.gz file

Design Grid.

This helper display a grid over the application to set better UI margin. On App.xaml.cs initialization simply add this line :

DesignGridHelper.setVisible(bool);    // activate grid
DesignGridHelper.Color = Colors.Red;  // set grid color
DesignGridHelper.Opacity = 0.7;       // set opacity between 0-1

Navigation

Two methods to navigate to page and purge backstack Use like this :

NavigationHelper.Navigate(string uri);
NavigationHelper.PurgeNavigationalBackstack(PhoneApplicationFrame rootFrame);

Validators

Regexp class helper to validate user input Validator types supported : Email, icq, creditcard number, english, url, IPv4, IPv6, name.

Usage : 

Validators.Validate(string inputText, validatorType type);

WebClient

Simple helper to GET/POST Data over HttpRequest

WebClientHelper.GET(WebClient wc, string url, Action<object, DownloadStringCompletedEventArgs> callback);

WebClientHelper.POST(WebClient wc, string url, string content, string contentType, Action<object, UploadStringCompletedEventArgs> callback);

MD5

MD5Core class for simple data encoding

byte[] GetHash(byte[] input);
string GetHashString(string input, Encoding encoding);
string GetHashString(byte[] input);
byte[] GetHash(string input);
byte[] GetHash(string input, Encoding encoding);
